# Madrasa Tus Salikat Lil Banat - Islamic Learning App

## Overview
A clean, premium, mobile-first Islamic Learning App with a beautiful green, white, and gold theme. This is a UI-only app (no backend, no Firebase, no authentication).

## App Structure
**Bottom Tab Navigation (5 tabs):**
- Home
- Courses
- Teachers
- Library
- About

## Screens

### Home Screen
- Arabic greeting "السلام عليكم"
- Madrasa name "Madrasa Tus Salikat Lil Banat"
- Featured Courses (horizontal scroll, 5 courses)
- Teachers Preview (3 teacher avatars)
- Announcements section (enrollment card)
- Quick Stats (14 Courses, 3 Teachers, 100+ Students)

6. Qirat Course
7. Bahare Shariat Course
8. Farze Uloom
9. Muballigah Course
10. Deeniyat Course
11. Tafseer Course
12. Urdu Course

- Sahih Al-Bukhari, Tafseer Ibn Kathir, Bahare Shariat, Fazail-e-Amaal, Nurul Idah, Riyad-us-Saliheen, Mishkat al-Masabih, Ihya Ulum al-Din

### About Screen
- Bismillah section
- Introduction

- Mission (5 bullet points)
- Contact placeholder

## Tech Stack
- Expo SDK 54 (React Native)
- Expo Router (file-based tab navigation)
- expo-linear-gradient
- @expo/vector-icons (Ionicons)

## Design System
- Primary: #0F3822 (dark green)
- Secondary: #D4AF37 (gold)
- Background: #FDFBF7 (warm white)
- Rounded cards, soft shadows, clean spacing
